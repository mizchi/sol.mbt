# Getting Started

Get up and running with Luna in minutes.

## Quick Start

The fastest way to get started is using the CLI:

### TypeScript/JSX

```bash
npx @luna_ui/luna new myapp
cd myapp
npm install
npm run dev
```

### MoonBit

```bash
npx @luna_ui/luna new myapp --mbt
cd myapp
moon update
npm install
npm run dev
```
