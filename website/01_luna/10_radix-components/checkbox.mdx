## Usage

```moonbit
let accepted = @signal.create(false)

radix_checkbox(
  checked=if accepted.get() { Checked } else { Unchecked },
  on_change=fn(v) { accepted.set(v == Checked) },
  children=[@luna.text("Accept terms")],
)
```

## With Label

```moonbit
@luna.div([class_("flex items-center gap-2")], [
  radix_checkbox(id="terms", checked=Checked),
  radix_label(for_id="terms", children=[@luna.text("Accept terms and conditions")]),
])
```

## States

- **Checked** - The checkbox is selected
- **Unchecked** - The checkbox is not selected
- **Indeterminate** - Partial selection (e.g., "select all" with some items selected)

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| checked | CheckboxState | Unchecked | Current state |
| disabled | Bool | false | Disable interaction |
| on_change | (CheckboxState) -> Unit | - | Change handler |

## Accessibility

- Uses `role="checkbox"` for screen readers
- Supports `aria-checked` with "true", "false", or "mixed"
- Keyboard accessible (Space to toggle)
- Focus visible indicator
