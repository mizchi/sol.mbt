## Usage

```moonbit
let volume = @signal.create(50)

radix_slider(
  value=volume.get(),
  min=0,
  max=100,
  on_change=fn(v) { volume.set(v) },
  children=[
    radix_slider_track(children=[
      radix_slider_range(value=volume.get()),
    ]),
    radix_slider_thumb(value=volume.get()),
  ],
)
```

## With Step

```moonbit
radix_slider(
  value=50,
  min=0,
  max=100,
  step=10,  // Move in increments of 10
  children=[
    radix_slider_track(children=[radix_slider_range(value=50)]),
    radix_slider_thumb(value=50),
  ],
)
```

## Props

### Slider

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | Int | min | Current value |
| min | Int | 0 | Minimum value |
| max | Int | 100 | Maximum value |
| step | Int | 1 | Step increment |
| orientation | SliderOrientation | Horizontal | Slider direction |
| disabled | Bool | false | Disable interaction |

## Accessibility

- Uses `role="slider"` for screen readers
- `aria-valuenow`, `aria-valuemin`, `aria-valuemax` for value range
- Arrow keys adjust value (Left/Down: decrease, Right/Up: increase)
- Focus visible on thumb
- Touch support for mobile devices
