## Usage

```moonbit
let selected = @signal.create("free")

radix_radio_group(
  name="plan",
  value=selected.get(),
  options=[
    radio_option("free", "Free Plan"),
    radio_option("pro", "Pro Plan"),
    radio_option("enterprise", "Enterprise"),
  ],
)
```

## Horizontal Layout

```moonbit
radix_radio_group(
  name="size",
  orientation=Horizontal,
  options=[
    radio_option("sm", "Small"),
    radio_option("md", "Medium"),
    radio_option("lg", "Large"),
  ],
)
```

## Props

### RadioGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| name | String | - | Group name for form submission |
| value | String? | None | Currently selected value |
| orientation | Orientation | Vertical | Layout direction |
| disabled | Bool | false | Disable all options |
| required | Bool | false | Require selection |

### RadioOption

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | String | - | Option value |
| label | String | - | Display label |
| disabled | Bool | false | Disable this option |

## Accessibility

- Uses `role="radiogroup"` for the container
- Arrow keys navigate between options
- Space selects the focused option
- Proper ARIA attributes for screen readers
