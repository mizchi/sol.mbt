## Usage

```moonbit
radix_progress(
  value=75,
  max=100,
  children=[radix_progress_indicator(value=75, max=100)],
)
```

## With Styled Version

```moonbit
// Uses Declarative Shadow DOM with inline styles
radix_progress_styled(value=50, max=100)
```

## Custom Styling

```moonbit
radix_progress_ucss(
  value=75,
  children=[radix_progress_indicator_ucss(value=75)],
)
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | Int | 0 | Current progress value |
| max | Int | 100 | Maximum progress value |

## Accessibility

- Uses `role="progressbar"` for screen readers
- `aria-valuenow` reflects current value
- `aria-valuemin` and `aria-valuemax` define range
- Smooth transitions for visual feedback
