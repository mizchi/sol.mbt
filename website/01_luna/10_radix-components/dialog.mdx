## Usage

```moonbit
let is_open = @signal.create(false)

radix_dialog(
  open=is_open.get(),
  children=[
    radix_dialog_trigger(children=[@luna.text("Open Dialog")]),
    radix_dialog_overlay(),
    radix_dialog_content(children=[
      radix_dialog_header(children=[
        radix_dialog_title(children=[@luna.text("Edit Profile")]),
        radix_dialog_description(children=[
          @luna.text("Make changes to your profile."),
        ]),
      ]),
      // ... form content
      radix_dialog_footer(children=[
        radix_dialog_close(children=[@luna.text("Cancel")]),
        radix_button(children=[@luna.text("Save")]),
      ]),
    ]),
  ],
)
```

## Parts

- **Dialog** - Root container with open state
- **DialogTrigger** - Button to open the dialog
- **DialogOverlay** - Background overlay
- **DialogContent** - Main content container
- **DialogHeader** - Header section
- **DialogTitle** - Dialog title
- **DialogDescription** - Description text
- **DialogFooter** - Footer with actions
- **DialogClose** - Close button

## Props

### Dialog

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| open | Bool | false | Whether dialog is open |

## Accessibility

- Uses `role="dialog"` and `aria-modal="true"`
- Focus trap within dialog content
- Escape key closes dialog
- Click outside (on overlay) closes dialog
- Focus returns to trigger on close
