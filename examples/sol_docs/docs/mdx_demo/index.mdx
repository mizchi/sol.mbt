# MDX Demo

This page demonstrates **MDX support** in Astra SSG. MDX allows you to use JSX components directly in your Markdown content.

## Interactive Counter

JSX components are automatically converted to Web Components:

<Counter initial="5" />

Web Components can also be used directly:

<luna-counter initial="10"></luna-counter>

## CLS Optimization

Prevent layout shifts with `cls:width` and `cls:height` hints:

<Counter initial="0" cls:width="150px" cls:height="50px" />

## Code Example

```javascript
// Define a Web Component
class LunaCounter extends HTMLElement {
  connectedCallback() {
    this.render();
  }
}
customElements.define('luna-counter', LunaCounter);
```
