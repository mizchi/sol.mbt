# About

This demo showcases the Luna framework's **Unified Progressive Architecture**.

## What is Luna?

Luna is a UI library for MoonBit that provides:

1. **Island Architecture** - Only hydrate interactive parts
2. **Signal-based Reactivity** - Fine-grained updates
3. **SSR + Hydration** - Server render with client interactivity
